Beauty Exceptions
=============

Installation
-------

```bash
composer require goenitz/beauty-exceptions
```
Usage
----------

```php
//error_reporting(E_ALL || ~E_NOTICE);
require 'vendor/autoload.php';

\Goenitz\BeautyExceptions\BeautyExceptions::register();
```

Ok, that's it!
